Human-Computer Interaction

Designing intuitive interfaces and understanding user behavior.

Overview

My research in Human-Computer Interaction focuses on creating seamless and intuitive digital environments. I explore how users interact with novel interfaces and how designing with empathy can drastically improve the user experience.

Key Projects & Explorations

  • Adaptive User Interfaces: Investigating how interfaces can dynamically adapt to a user's context and cognitive load.
  • Multimodal Interaction: Combining voice, gesture, and touch to create more expressive computing experiences.
  • Accessibility: Designing systems that cater to users with diverse abilities.
